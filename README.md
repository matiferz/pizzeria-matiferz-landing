#Pizzería Matiferz

Landing page responsive para una pizzería ficticia, desarrollada como proyecto de práctica para mejorar habilidades en desarrollo web frontend.

#Tecnologías utilizadas

- HTML5
- CSS3 (Flexbox, diseño responsive)


#Características

- Diseño landing page
- Navegación con scroll entre secciones
- Secciones incluidas:
  - Hero principal
  - Menú (pizzas y empanadas)
  - Nosotros
  - Contacto con formulario
  - Footer
- Diseño responsive para distintos dispositivos
- Uso de imágenes personalizadas

#Objetivo del proyecto

El objetivo de este proyecto fue practicar:

- Estructura semántica en HTML
- Maquetado con CSS
- Uso de Flexbox
- Diseño responsive
- Organización de un proyecto real
